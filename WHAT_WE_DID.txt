STEP 2 - We implemented the algorithm in Python
------------------------------------------------
We created 4 Python files:

  count_min_sketch.py
    This is the memory structure MIDAS uses.
    It counts how many times each edge appeared
    without storing every edge.

  midas.py
    The basic MIDAS algorithm.
    For each edge (u, v, t) it computes a score:
    score = (current count - expected count)^2 / expected count
    High score = anomaly (sudden burst)

  midas_r.py
    Improved version of MIDAS.
    Also tracks node-level counts (not just edge counts).
    Uses decay so recent edges matter more.
    Gets higher AUC than basic MIDAS.

  sedanspot.py
    The baseline algorithm from the paper.
    Uses random walks to score edges.
    Much slower than MIDAS.


STEP 3 - We ran on DARPA dataset
---------------------------------
DARPA has 4.5 million network connections.
Some are attacks, some are normal.
We ran all 3 algorithms and got:
  MIDAS   AUC = 0.87  (paper says 0.91)
  MIDAS-R AUC = 0.96  (paper says 0.95)
  SedanSpot AUC = 0.60  (paper says 0.64)
Close to the paper. Small difference because Python vs C++.

STEP 5 - We applied MIDAS to new datasets
------------------------------------------
Professor asked us to try MIDAS on different data.
We used:

  Primary School dataset (125,773 contacts between children)
    - Each contact is an edge: child A touched child B at time T
    - We defined anomaly = children from DIFFERENT classes touching
    - MIDAS AUC = 0.65 (detects cross-class bursts during breaks)
    - Used metadata file to add gender info to results

  Hospital dataset (32,424 contacts between patients and staff)
    - Each contact is an edge: person A touched person B at time T
    - We defined anomaly = patient touching another patient
    - MIDAS AUC = 0.75 (detects unusual patient-patient bursts)
