# Hospital Ward Dynamic Contact Network

Download from: https://sociopatterns.org/datasets/hospital-ward-dynamic-contact-network/

Place this file here:
- `detailed_list_of_contacts_Hospital.dat` — 32,424 rows, tab-separated: t i j Si Sj

Format:
- t  = timestamp in seconds (20-second intervals)
- i,j = anonymous node IDs
- Si,Sj = roles: PAT (patient), MED (doctor), NUR (nurse), ADM (admin)
